Vent Generator


View on Heroku / / View on Github


A relatively simple exercise in AJAX calls, this program is a random phrase generator. Utilizing a when/then in jQuery, once all AJAX calls are complete the new random phrase is appended to the DOM. An amusing addition to the project is displayed underneath the random phrase where I used a conditional, checking for a vowel, to determine if 'a' or 'an' is to be used.
