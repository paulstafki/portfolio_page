Message Board


View on Heroku / / View on Github


An exercise in routing practices within Node and ExpressJS, along with an intro to Mongoose. A message board that takes a name and text, then writes both to the MongoDB database. Upon success, all posts are retrieved and the DOM is updated. This app also serves up a secret route, which adds a delete button to each post for the UX.
